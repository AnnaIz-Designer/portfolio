03 · Process

0 → 1 in steps.

01
Research — parcel anxiety loop

Users track parcels multiple times a day out of anxiety. The app needed to reduce that loop — clear status, proactive notifications, one-glance clarity.

02
Information architecture

Mapped all flows: add parcel, track status, manage archive, subscribe, set reminders. Defined tab structure and navigation for both platforms.

03
Token-based design system in Figma

Semantic token system from scratch using Figma Variables — colors, spacing, typography, states. Separate files for iOS (HIG) and Android (Material 3).

04
All screens · 2 platforms · 2 modes

Every screen in light and dark mode for both iOS and Android. Platform conventions respected while keeping visual language consistent.

05
Paywall, warranty & reminder UX

Monetization flow — free trial, annual vs weekly plans. Plus warranty tracker and return window reminders as key differentiators.
